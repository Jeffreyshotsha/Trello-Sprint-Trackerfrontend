# Trello Sprint Tracker Frontend

Modern React + Vite app for tracking Trello sprints.

## Features
- Current 7-day sprint burndown chart
- Story points & team member breakdown
- Sprint history view
- Persistent Board ID (saved in localStorage)
- Progress bar based on completed points
- Personal About page with founder story

## Tech Stack
- React + Vite
- Tailwind CSS
- Chart.js
- Day.js
- Lucide icons
- React Router

## Setup
```bash
npm install
npm run dev
